# v-ecomm
# `UptoHeroku`
```bash
heroku login
heroku git:remote -a v-ecomm
git push heroku main
```
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
